Anthropicが2025年10月20日、ブラウザからClaude Codeにコーディングタスクを委任できる「Claude Code on the web」を研究プレビューとして公開しました。ローカルのターミナルを開かなくても、Anthropicのクラウドインフラ上で複数のコーディングタスクを並行して走らせ、進捗を確認しながら管理できるのが特徴です。

## 詳細

- **並行タスク実行**: 異なるリポジトリにまたがる複数のコーディングセッションを、1つの画面から同時に起動・管理できる
- **進捗のリアルタイム確認**: Claudeの作業状況を随時確認し、必要に応じて方向修正の指示を出せる
- **Pull Requestの自動作成**: 変更内容の要約付きでPRを自動生成し、レビューへの動線をつなぐ
- **モバイル対応**: iOSアプリでの早期プレビューも同時提供
- **想定用途**: バグ修正や定型的なメンテナンス、テスト駆動開発を伴うバックエンド変更など
- **サンドボックス環境**: タスクごとに隔離されたサンドボックスを用意し、ネットワーク・ファイルシステムへのアクセスを制限。Gitとのやり取りは認可済みリポジトリのみにアクセスできるセキュアなプロキシ経由で行われる
- **提供範囲**: 当初はPro・Maxプランの利用者向け。claude.com/codeからGitHubリポジトリを連携して利用開始でき、利用量は他のClaude Code利用分とレート制限を共有する

## その後

2025年11月12日には、Team・Enterprise向けのプレミアムシート利用者にも提供範囲が拡大。ローカル端末に縛られないエージェント型コーディングという方向性は、その後のバックグラウンドエージェント機能群の土台となっていく。
