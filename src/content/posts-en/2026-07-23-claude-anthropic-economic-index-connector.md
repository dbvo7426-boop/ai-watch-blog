Anthropic has launched the Anthropic Economic Index connector, letting users explore Anthropic Economic Index data directly within Claude conversations. Anyone can enable it in about a minute to explore how AI is actually being used across the economy in a conversational format.

## Details

- **Data provided**: AI usage patterns by occupation and industry, regional variations in Claude usage (such as Colorado-specific data), task automation trends over time, and how specific professions like teachers use Claude
- **Sample questions**: It can answer things like "Which occupations use AI the most?", "What are the most common ways people in Colorado use Claude?", "What sorts of tasks do teachers use Claude for?", and "What kinds of tasks are people automating with AI?"
- **How to use it**: Open the connectors menu in claude.ai, find the Anthropic Economic Index, and enable it — takes about a minute. It works with any Claude model in any conversation and requires no installation
- **A key limitation**: Anthropic notes that "the Index reflects patterns in Claude usage rather than the labor market as a whole," and Claude points users back to the source data and its limitations during exploration
- **Availability**: Launched July 22, 2026, on claude.ai, with the full underlying datasets freely available on Anthropic's website

## How to try it

Log into claude.ai, search for "Anthropic Economic Index" in the connectors menu, and enable it to start using it immediately. No extra fee or special plan is required.
