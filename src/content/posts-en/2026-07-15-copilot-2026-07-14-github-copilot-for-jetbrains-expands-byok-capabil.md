GitHub has updated GitHub Copilot for JetBrains with major improvements to model/provider flexibility and customization.

## Details

- **Model and provider flexibility**: Now supports BYOK (Bring Your Own Key), letting users configure OpenAI-compatible custom endpoints with an API key to use their own models. Claude agent provider customization also allows configuring custom agents and instructions
- **Enhanced development workflow**: A built-in debugger skill for Copilot CLI enables agent-driven debugging directly within the development environment. Plugin management has also been expanded through the marketplace and source repository browsing
- **Local sandbox**: Now available as a public preview
- **Other improvements**: Better model selection UX, message re-editing, and improved reliability for auth recovery, account switching, provider/session persistence, and editor integration

## How to try it

| Feature | Eligible plans |
| --- | --- |
| Claude agent provider customization | GitHub Copilot Pro and above (public preview) |
| BYOK, debugger skill, and other features | All subscription tiers |

The latest version is available immediately from the JetBrains Plugin Marketplace.
