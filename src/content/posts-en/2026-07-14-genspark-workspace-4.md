Genspark unveiled "AI Workspace 4.0." Built around the concept of "your AI employee," it integrates AI agents across multiple work environments.

## Details

- **Genspark Claw for Desktop**: A native desktop client featuring "Computer Use," which manages local files, apps, and screen operations, and "Browser Use," which automates web navigation and data collection
- **Microsoft Office integration**: Built in as native plugins for PowerPoint, Excel, and Word, providing AI assistance without leaving the app
- **Speakly meeting features**: "Live Translation" for real-time multilingual translation, and "AI Meeting Notes," which automatically joins scheduled calls and generates summaries
- **Advanced workflows**: A high-speed execution engine built on OpenCode handles complex, multi-step automation tasks

## How to try it

Because it's designed to embed AI agents directly into the tools you already use — Office, desktop, and meeting apps — there's no need to change your workflow. You can hand off mechanical work like file management, web research, data analysis, and note-taking to AI. No specific pricing plan was stated, but you can try it from the official site.
