Google has announced that it is renaming "NotebookLM" to "Gemini Notebook." While it remains positioned as a standalone product, the rename is meant to deepen its ties with the broader Gemini ecosystem, and comes alongside a new feature that lets users write and run code within each notebook.

## Details

- **Reason for the rename**: Positioned as continuing to be "a research tool as a standalone product, while being able to do more across Google's broader ecosystem"
- **Code execution feature**: Each notebook now has a secure cloud execution environment where you can write and run Python code for complex data analysis based on your source materials. Currently available to **Google AI Ultra** subscribers and some Workspace customers, with a rollout to all Pro users planned over the coming weeks
- **Deeper integration**: The Gemini app will sync with notebooks, and Google Search's "AI Mode" is planned to gain direct access to research materials as well
- **What stays the same**: The core mission since its 2023 launch as "Project Tailwind" — being a research tool that streamlines learning and work — continues unchanged
- **Usage stats**: Google says it has over 30 million users, with more than 600 organizations using it to improve productivity
- **Rollout**: The name and logo change will roll out across interfaces over the coming weeks. Existing shared notebooks and links will redirect automatically, with no action needed from admins (mobile apps may require a manual update)

## How to try it

- Existing NotebookLM users can continue using the same account seamlessly as "Gemini Notebook" (automatic migration, no additional setup needed)
- The code execution feature is available to Google AI Ultra subscribers, or Workspace customers with appropriate access, with a broader rollout to follow
