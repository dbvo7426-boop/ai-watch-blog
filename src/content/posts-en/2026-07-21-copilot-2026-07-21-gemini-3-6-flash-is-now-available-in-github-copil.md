GitHub has made Google's latest model, Gemini 3.6 Flash, available in GitHub Copilot. The model is designed for web and app development, coding, and longer-horizon agentic tasks, with configurable reasoning effort and support for parallel tool use. Early testing showed it delivers better task completion and token efficiency than its predecessor, Gemini 3.5 Flash.

## Details

- **Eligible plans**: Copilot Pro, Pro+, Max, Business, and Enterprise
- **Supported surfaces**: Visual Studio Code, Visual Studio, Copilot CLI, the GitHub Copilot cloud agent, the GitHub Copilot app, JetBrains, Xcode, and Eclipse
- **Design intent**: Optimized for web/app development, coding, and long-horizon agentic tasks, with configurable reasoning effort and parallel tool calling
- **Performance**: Early tests showed better task completion and token efficiency compared to the prior Gemini 3.5 Flash
- **Rollout**: Being rolled out gradually. Copilot Enterprise and Business administrators need to enable the "Gemini 3.6 Flash Preview" policy in their organization's Copilot settings before users can select it
- **Billing**: Billed at Google's provider list pricing through usage-based billing

## How to try it

- On an eligible plan where the rollout has reached you, select Gemini 3.6 Flash from the model picker in any supported editor or app
- Business/Enterprise administrators must first enable the "Gemini 3.6 Flash Preview" policy in their organization's Copilot settings
