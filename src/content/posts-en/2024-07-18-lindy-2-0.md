Lindy released the major update "Lindy 2.0." Its centerpiece is the "Flow Editor," an AI-assisted flowchart-style editor designed to boost agent reliability while preserving flexibility and an AI-native feel.

## Details

- **Flow Editor**: A new editor that lets users assemble steps like a flowchart. Beta testers reportedly built agents that were far more sophisticated and reliable than before
- **Data referencing**: Later steps can now reference data from earlier triggers and actions — for example, "use a Slack message as a trigger, then create a Linear ticket based on its content"
- **WhatsApp Business integration**: Added support for messaging via WhatsApp Business, which beta users applied to a wide range of automations from customer support to sales inquiries
- **Multiple account support**: Users can now handle multiple accounts for the same integration (e.g., Gmail), enabling workflows like sorting email across several inboxes
- **Other additions**: Sticky notes, plus support for GPT-4o mini and Claude 3.5 Sonnet

## What happened next

After Lindy 2.0, Lindy kept adding incremental features, expanding the platform with phone call support (January 2025) and code execution actions (May 2025). This led into "Lindy 3.0" in August 2025, which introduced the next vision: the "AI employee."
