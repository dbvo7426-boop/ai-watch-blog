On July 7, 2026, Anthropic announced the public beta launch of Claude Code and Claude Cowork (its knowledge-work agent) on "Claude for Government Desktop," a FedRAMP High-certified environment. Public-sector software teams can now use the coding agent, and the launch also brings government-oriented controls such as department-level administration and spend management.

## Details

- **Claude Code for Government**: Offered to public-sector software teams as an AI coding agent for building and modernizing the software systems that support public services
- **Claude Cowork**: An agent that lets staff delegate knowledge work — drafting memos, reviewing RFPs (requests for proposals), documenting casework, and building slide decks — directly from the desktop
- **FedRAMP High-certified environment**: Delivered within the certified "Claude for Government Desktop" environment, meeting government agencies' security requirements
- **Government-oriented governance features**: Also includes local conversation history retention on agency-managed devices, department-level administration, usage and model limits, tamper-evident audit logs, and authorization-related documentation
- **Availability**: Launching as a public beta. New customers can request access at claude.com/solutions/government

## How to try it

- Intended for government (public-sector) staff and procurement officials
- Access is requested at claude.com/solutions/government, followed by an agency-level onboarding process
- This is a separate offering from the general individual developer plans, so it has no direct impact on regular Claude Code (Pro/Max/Team/Enterprise) users
