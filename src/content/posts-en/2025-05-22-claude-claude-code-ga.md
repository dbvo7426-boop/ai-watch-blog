On May 22, 2025, alongside the announcement of its new models "Claude Opus 4" and "Claude Sonnet 4," Anthropic moved "Claude Code" — which had spent about three months in a limited research preview — to general availability (GA). Beta extensions for VS Code and JetBrains, GitHub Actions integration, and a developer SDK were all released the same day, expanding its reach beyond the terminal.

## Details

- **Move to general availability (GA)**: Following positive feedback during the research preview period, the sign-up requirement was lifted, letting developers start using Claude Code freely
- **VS Code and JetBrains extensions (beta)**: Edits proposed by Claude now appear inline within the editor, letting review and tracking happen entirely inside a familiar IDE. Installed by launching Claude Code from the IDE's terminal
- **Background tasks via GitHub Actions integration**: Time-consuming tasks can now run in the background without blocking the conversation
- **GitHub integration (beta)**: The `/install-github-app` command lets users tag Claude Code on a pull request, delegating tasks such as addressing review comments, fixing CI errors, and making code changes
- **Claude Code SDK released**: Also released an extensible SDK, built on the same core technology as Claude Code itself, for building custom agents
- **Claude 4, announced alongside**: Announced Claude Opus 4, strong at long-running autonomous work, and Claude Sonnet 4, suited to everyday use. Both models were said to set new standards in coding and agentic tasks

## What happened next

About four months after GA, in September 2025, a major update added subagents, hooks, and a checkpoint (rewind) feature, further strengthening autonomy, and two months after that, "Claude Code on the web," usable from a browser, arrived as well. Within roughly half a year of reaching GA, annual recurring revenue (ARR) hit the $1 billion mark, and it grew rapidly into the core of Anthropic's agentic product strategy.
