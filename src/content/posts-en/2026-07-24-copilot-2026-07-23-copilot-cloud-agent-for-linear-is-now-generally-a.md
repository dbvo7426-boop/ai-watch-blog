GitHub has announced that its Copilot cloud agent integration for Linear is now generally available, letting teams assign Linear issues directly to the Copilot cloud agent. The Copilot cloud agent is an asynchronous, autonomous background agent that analyzes an issue and works on it inside an isolated, ephemeral environment powered by GitHub Actions, without requiring constant oversight.

## Details

- **How it works**: Assign a Linear issue to the agent, and it analyzes the contents and opens a draft pull request from within its isolated environment
- **Progress updates**: Streams progress to Linear's activity timeline as it works
- **On completion**: Requests a pull request review once the work is done
- **Team controls**: Choose a model per task, deploy custom agents from a repository, specify target and working branches, and adjust instructions in real time via comments
- **Eligible plans**: Copilot Pro, Pro+, Business, and Enterprise
- **Availability**: Generally available as of July 23, 2026

## How to try it

- Install the "GitHub Copilot for Linear" app from the GitHub Marketplace (requires GitHub organization owner permissions and Linear workspace admin privileges)
- Follow Linear's agent configuration documentation to finish setup
