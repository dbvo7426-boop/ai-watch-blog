Manus has introduced a new feature called "Plan Mode." Before the AI agent starts working, it now presents a structured, Markdown-formatted plan that users can review and edit, then approve for execution. The feature grew out of the observation that "most AI mistakes happen because no one reviewed the approach" beforehand.

## Details

- **The problem it addresses**: Previously, misalignment between user intent and AI execution often surfaced only after a task was already finished, requiring extensive rework
- **How to activate it**: On web, type "/" to bring up the command menu; on mobile, tap the "+" menu, then select Plan Mode
- **The workflow**: After activation, answer any clarifying questions Manus asks, review the generated Markdown plan, edit it as needed, and approve — Manus then builds exactly to the approved plan
- **Flexibility**: Activation is manual (it never triggers automatically), can be turned on mid-task, and the plan can be edited multiple times before approval
- **Intended use cases**: Designed with development scenarios and edge-case detection in mind, reducing the need for heavily engineered prompts

## How to try it

- Available now: type "/" on web or tap "+" on mobile and select Plan Mode
- No extra fee is mentioned — it's rolled out to existing Manus plan users as-is
